# Poco M2 Pro ROM Changelog

## Version 1.0.0 - Initial Release
- Initial release of the ROM for Poco M2 Pro.
- Base features: customizations, stability improvements.
- Initial integration with latest Android security updates.

## Version 1.1.0 - Bug Fixes and Improvements
- Resolved issues with system lag.
- Improved battery life.
- Added new customization options in settings.
- Minor UI tweaks and enhancements.

## Version 1.2.0 - Feature Update
- Introduced new themes and wallpapers.
- Enhanced performance with optimized kernel.
- Added support for advanced camera features.
- Improved app launch speed and responsiveness.

## Version 1.3.0 - Major Update
- Added support for the latest Android version.
- Comprehensive UI redesign with modern elements.
- New features: split-screen multitasking, improved notifications.
- Fixed known issues and bugs from previous versions.
